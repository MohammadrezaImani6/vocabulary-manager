# Vocabulary Manager 📚

A Python command-line app to manage English vocabulary words.

## Features

- ➕ Add new words with meaning and example
- 🔍 Search words
- ✏️ Edit words
- 🗑️ Delete words
- 📋 Show all words
- 🎯 Quiz yourself
- 📊 Statistics
- 💾 Auto-save to JSON and TXT

## Requirements

- Python 3.x

## How to run

```bash
python main.py
```

- Python
